# Clustering-Based-Edge-Augmentation-for-Minimizing-the-Kirchhoff-Index
Experiments for the AI STATS 26 Submission of Clustering-Based Edge Augmentation for Minimizing the Kirchhoff Index


Instructions for running code:-

1. There are three Jupyter notebooks in this folder that run
   
     a) Greedy algorithm 
 
     b)2K-median_plus_Greedy algorithm
 
     c)K_median_penalties_from__single_source


3. There are various graph datasets in the "data" folder that contain the various text files, where each text file corresponds to a particular graph that is used in each table of the experiments in Section 6


4. Each text file contains multiple rows, where each row represents an edge in that graph.
 
5. To run the code for each data set or graph, we just need to keep the text file corresponding to that graph in the same folder as the Jupyter notebook and run the code.
